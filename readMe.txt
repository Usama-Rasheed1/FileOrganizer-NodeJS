File Organizer

File Organizer is a Node.js script that organizes files in a directory by moving them into subdirectories based on their file extensions.

Features

1. Automatically creates folders for different file types (e.g., `js`, `txt`, `pdf`) if they don't exist.
2. Moves files into respective folders based on their extensions.
3. Skips organizing itself (`fileOrganizer.js`).
